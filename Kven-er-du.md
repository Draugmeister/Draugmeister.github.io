# Kven er du?

- Trill 2d6 og legg saman resultatet for å finna bakgrunnen din.

- Trill fram eit namn, eller finn på eit sjølv.

- Fordel evnene dine:
  +2, +1, +1, 0, 0, -1

- Noter ned livskrafta di (10 +CON).

- Noter ned bæreevne (10 +STR).
